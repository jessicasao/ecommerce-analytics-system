## 📖 專案簡介

這是一個專為電商品牌設計的**數據自動化系統**，整合 **Shopify** 和 **Pinkoi** 雙渠道訂單數據，自動生成月度財務報表與客戶分析。解決手動整理數據耗時、容易出錯的問題，並提供商業洞察。

🔍 **為什麼要做這個？**
- 每月手動整合兩平台數據需 5-8 小時
- 折扣分攤不精準，影響利潤計算
- 無法即時掌握客戶貢獻度和渠道效益

---

## ✨ 核心功能

| 功能模塊 | 說明 |
|---------|------|
| **📥 多源數據整合** | 自動標準化 Shopify 和 Pinkoi 的不同欄位結構 |
| **💰 精準利潤計算** | 按比例分攤訂單折扣到每個商品，計算真實利潤 |
| **👥 客戶行為分析** | 重複購買率、客單價、客戶貢獻度排行 |
| **📊 渠道效益評估** | Shopify vs Pinkoi ROI 分析、渠道佔比 |
| **📑 自動化報表** | 一鍵生成 Excel 月度財務報表（4 個工作表） |
